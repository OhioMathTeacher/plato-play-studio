  • Quick check: How's Problem Set 1 going? Any misconceptions we should clear up before moving on?
2. Read Together Using Close Reader (20 minutes: 8:40-9:00)

Open Close Reader & Explain Tools (5 min)

  • Teacher: open Close Reader: Plato's Allegory of the Cave on the projector
  • Students: open the same link on their own devices
  • Take a moment to explore on your own: hover over each feature to see what it does
  • Point out the key tools:
    • Text panel (Sheehan translation, already divided into 3 parts)
    • Language toggle—switch between English and Modern Greek to see your skia / alētheia / paideia words in context
    • Margin notes—click any passage, add a note, it auto-saves to your browser
    • AI annotations—select a passage, ask a question. The system uses an API KEY to connect to an AI provider that responds to your specific question about that text. This is a tool, not a substitute for your thinking. You can also use "No AI" mode and still get full margin notes.

Shared Reading (15 min)

  • Read all 3 parts aloud as a class, following along in Close Reader (Sheehan translation)
  • Options for reading:
    • Assign parts: Socrates and Glaucon
    • Or rotate readers every paragraph
  • Don't rush—Sheehan already divides it into 3 clean sections
  • Timed pauses for engagement: After each section, pause for 30-60 seconds and let students add a margin note or ask a question using AI annotations. This keeps the text personal.
  • Model the close-reading process: when a good question emerges naturally, pause and use the AI annotation tool to dig deeper. Show students how to ask a specific, useful question.
  • As you listen, drop margin notes whenever you spot skia, alētheia, or paideia in the text—connects directly to last night's homework
3. From Dialogue to Drama (30 minutes: 9:00-9:30)

The Key Question (5 min)

  • Sheehan already split this into 3 parts for us.
  • So is this ready to perform? Could we just read it aloud on stage?
  • Wait for responses...
  • What's MISSING?

Brainstorm on Board (20 min)

  • What do we need to ADD to make this a play?
  • Collect ideas on the board
  • This is why there are almost NO theatrical adaptations of the Allegory—it's a dialogue ABOUT a scenario, not a script.
  • YOUR job is to create what's missing.

Discuss the Scenes (5 min)

  • Briefly review what happens in each part:
    • Part 1 (Scene 1): The Cave + Liberation — prisoners, shadows, one is freed
    • Part 2 (Scene 2): The Ascent — painful journey up, blinded by light, seeing truth
    • Part 3 (Scene 3): The Return — going back, trying to help, being rejected
4. Assign Groups, Introduce the Studio & Start Drafting (20 minutes: 9:30-9:50)

Form Groups (3 min)

  • Announce group assignments: & Introduce the Studio (10 minutes: 9:30-9:40)

    Form Groups (3 min)

    • Announce group assignments:
    Group Members Task
    Triad A [names] Scene 1: The Cave + Liberation
    Triad B [names] Scene 2: The Ascent
    Triad C [names] Scene 3: The Return
    Pair D [names] Production Crew (set, props, narrator)
  • Groups sit together

Introduce Plato Play Draft Studio (7 min)

  • Open Plato Play Draft Studio on the projector
  • This tool will pull each group's Google Doc and format it as a script—automatically. On Day 19 we'll use it to combine all four drafts into one script for the table read.
  • Quick demo: Group Setup → paste a Google Doc link → Pull Doc → see the formatted preview instantly
  • Script Creation Tips: Give each student the "Script Creation Tips" doc. This shows the difference between what Plato wrote and what you need to create.
  • For the studio to format your draft correctly, use these conventions in your Google Doc:
    What you want What to type in your Google Doc
    Character dialogue SOCRATES: Your line here.
    or  @Prisoner@ Your line here.
    Stage direction (Lights dim. A single torch flickers.)
    Scene heading [[[Scene 1 — The Cave]]]
    Pair D cue markers [[OPENING]]   [[TRANSITION_2]]   [[TRANSITION_3]]   [[CLOSING]]
5. Work Time (10 minutes: 9:40-9:50)

Open Your Google Docs & Get Started

  • Each group opens their shared Google Doc and sets up the Group Setup entry in the studio (label, scene description, Google Doc link)
  • Scene Groups:
    • Re-read your section of the Allegory
    • What needs to be ADDED to make it performable?
    • Begin drafting: Start typing your scene using the markup format from above. Include character names, dialogue, and stage directions.
  • Production Crew:
    • How will we create the cave? The shadows? The sun?
    • What will the narrator say to frame each scene? Start drafting these as cue-marker sections ([[OPENING]], [[TRANSITION_2]], etc.)
    • Sketch staging ideas
  • Not ready to start drafting yet? Use this time to continue work on Problem Set 4 from earlier this week. You can jump into drafting when ready.
